## 应用特殊说明

没有说明的话，后台密码通常为 `admin:admin`；有的应用默认端口为 `8080、8161` 等等，已经全部改为 `80`

### MySQL

默认开启查询日志，aka `/var/log/mariadb/query.log`

### TPShop

后台地址

```
http://127.0.0.1/index.php/Admin/Index/index
```

### Tomcat

后台密码 `tomcat:tomcat`，监听端口改为 `80`

### Apache

默认开启 `/server-status`

### MetInfo

后台地址

```
http://127.0.0.1/admin/
```
